# installation_scripts
Repo for installation scripts for windows, mac & ubuntu


## For mac
```bash
chmod +x install-mac.sh
./install-mac.sh
```

## For windows (RUN POWERSHELL AS ADMIN)
```bash
Set-ExecutionPolicy Bypass -Scope Process
.\install-windows.ps1
```


## For ubuntu / debian
```bash
chmod +x install-debian.sh
./install-debian.sh
```
